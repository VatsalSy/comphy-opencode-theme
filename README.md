# OpenCode Comphy Gruvbox Theme

A warm, retro-inspired Gruvbox color theme for OpenCode IDE, optimized for the CoMPhy Lab workflow.

## Overview

This theme brings the beloved Gruvbox color scheme to OpenCode, featuring warm earth tones that reduce eye strain during long coding sessions. Carefully calibrated contrast levels ensure excellent readability for scientific computing and computational physics work.

## Features

- Warm Gruvbox color palette optimized for OpenCode
- Excellent contrast for code readability
- Carefully selected syntax highlighting for scientific computing
- Eye-friendly colors for extended coding sessions

## Installation

```bash
# Clone the repository
git clone https://github.com/VatsalSy/opencode-comphy-gruvbox.git

# Copy theme to OpenCode themes directory
cp opencode-comphy-gruvbox/theme.json ~/.config/opencode/themes/
```

## Usage

1. Open OpenCode settings
2. Navigate to Themes
3. Select "Comphy Gruvbox" from the theme list
